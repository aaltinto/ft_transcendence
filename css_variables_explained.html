CSS Variables Nasıl Çalışır?

1. Değişken Tanımlama:

--ana-bosluk: 20px; → Sadece değer sakla
--ana-renk: #3498db; → Sadece değer sakla

2. Değişken Kullanma:

margin-bottom: var(--ana-bosluk); → Burada söylüyorsun hangi özellikte kullanacağını!
color: var(--ana-renk); → Burada söylüyorsun hangi özellikte kullanacağını!
